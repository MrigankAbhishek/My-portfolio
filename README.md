# 🧑‍💻 Mrigank Abhishek — Personal Portfolio

A clean, minimal personal portfolio website built with React.

🌐 **Live Demo:** [your-portfolio.vercel.app](https://your-portfolio.vercel.app)

---

## ✨ Features

- Tabbed navigation — About, Education, Projects, Skills
- Demo links on projects
- Responsive layout — works on mobile and desktop
- Zero external dependencies — just React
- All data in one object at the top of `Portfolio.jsx` — easy to update

---

## 🛠 Tech Stack

| Layer | Tool |
|---|---|
| Frontend | React (Create React App) |
| Styling | Inline CSS (JS style objects) |
| Deployment | Vercel |
